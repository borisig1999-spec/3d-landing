Какие пластики бывают

Коротко о самых популярных — какой для чего подходит.

Базовый

PLA

Самый популярный. Легко печатать, яркие цвета, без запаха. Для декора, прототипов, игрушек.

Гибкий

TPU / Flex

Гибкие, прочные, не ломаются при изгибе. Для втулок, уплотнителей, защитных чехлов.

Прочный

ABS

Ударопрочный, термостойкий. Для автомобильных деталей, корпусов, функциональных узлов.

Инженерный

PA6 / Нейлон

Износостойкий, прочный. Для втулок, шестерён, деталей под нагрузкой.
